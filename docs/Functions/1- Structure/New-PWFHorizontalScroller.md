# New-PWFHorizontalScroller

## Description
Add an horizontal scroller of your next div. Use it for very large content.

## SYNTAX
``` powershell
New-PWFHorizontalScroller -Content <scriptblock>
```

## EXAMPLES

### EXAMPLE 1
```powershell
New-PWFHorizontalScroller -Content {
    ...SOMECODE...
}
```

## PARAMETERS
### -Content
Your next code.
