# New-PWFBlockQuote

## Description
Create a blockquote.

## SYNTAX
``` powershell
New-PWFBlockQuote -YourText <string>
```

## EXAMPLES

### EXAMPLE 1
```powershell
New-PWFBlockQuote -YourText "My text in a blockquote"
```

## PARAMETERS
### -YourText
Type the text you want to display in a blockquote.
