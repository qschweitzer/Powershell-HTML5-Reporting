# New-PWFText

## Description
Create a simple text div.

## SYNTAX
``` powershell
New-PWFText -YourText <string> [-Center <switch>]
```

## EXAMPLES

### EXAMPLE 1
```powershell
New-PWFText -YourText "My awesome text" -Center
```
### EXAMPLE 2
```powershell
New-PWFText -YourText "My awesome text"
```
